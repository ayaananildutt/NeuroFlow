Edge Layer

CCTV cameras feed video to a Raspberry Pi gateway running YOLOv8 for real-time vehicle detection at the intersection.

üì°

Network Layer

Lightweight MQTT protocol transmits JSON detection payloads from edge devices to the cloud with minimal latency.

‚òÅÔ∏è

Cloud Layer

FastAPI Traffic Controller processes data, persists to PostgreSQL, and pushes live updates to the React dashboard via WebSocket.

üîÅ

Feedback Loop

Server sends adaptive signal timing commands back to traffic lights, creating a closed-loop autonomous control system.

üìê Data Flow Pipeline

1

Video Capture

CCTV camera streams video ‚Üí OpenCV captures frames every 500ms on the Raspberry Pi edge gateway.

2

Vehicle Detection

YOLOv8 nano model runs inference on each frame. Detects and classifies: car truck bus motorcycle.

3

MQTT Publish

Detection results serialized as JSON ‚Üí published to neuroflow/detections/{intersection_id} with QoS 1.

4

Cloud Processing

Traffic Controller server receives detections ‚Üí persists to PostgreSQL ‚Üí computes 10-frame moving average density.

5

Adaptive Signal Timing

Green duration scales linearly with density: base + density √ó factor √ó base, clamped to 15s‚Äì90s.

6

Signal Command

Phase change published via MQTT to neuroflow/commands/{id} ‚Üí traffic light adjusts in real-time.
