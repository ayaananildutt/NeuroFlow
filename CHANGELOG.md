# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-14

### Added
- **Edge Layer**: CCTV camera feed capture with OpenCV and YOLOv8 vehicle detection.
- **MQTT Integration**: Lightweight pub/sub messaging between edge devices and cloud server.
- **Traffic Controller Server**: FastAPI-based server with adaptive signal timing algorithm.
- **PostgreSQL Logging**: Full detection and signal command history stored in PostgreSQL.
- **React Dashboard**: Real-time traffic visualization with intersection map, metrics charts, and signal override controls.
- **Docker Compose**: Full-stack orchestration with Mosquitto MQTT broker, PostgreSQL, server, and dashboard.
- **CI Pipeline**: GitHub Actions workflow for Python linting, JS linting, and Docker build validation.
- **Documentation**: Architecture diagrams (Mermaid.js), API reference, and setup guide.
